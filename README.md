# job4j_cinema
## Проект "Кинотеатр"
### Веб-приложение для покупки мест в кинотеатре
## Используемые технологии

* Java servlets
* JavaScript (jQuery, Ajax), HTML
* JDBC
* PostgreSQL
* Log4j, Sl4J
* Maven
* Apache Tomcat
* Travis CI, CheckStyle
